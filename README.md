# WorkSphere - Enterprise Employee Management System

## 📌 Project Overview

WorkSphere is a modern full-stack Employee Management System designed for organizations to manage employees, departments, tasks, attendance, leave requests, and performance.

This project is being developed as part of the **Advanced Software Engineering and Development Internship**.

---

## 🚀 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JWT
- bcrypt

### Deployment
- Vercel
- Render
